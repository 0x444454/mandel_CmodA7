# BITSTREAM IS HERE

- **top.bit** : Bitstream for Digilent Cmod A7-35T board.

NOTE: This bitstream is for the 35T (XC7A35T).  
If you have a 15T (XC7A15T), you must build from sources (see README file in src folder).

# HOW TO RUN

Use Vivado to program your Cmod A7-35T board using the bitstream file.  
In the "Flow Navigator": "Program and Device" -> "Open Hardware Manager" -> "Program Device" -> [your board].  
Select the path of the "top.bit" bitstream file.  
Click the "Program" button on the UI.

NOTE: If "Program Device" is not active, check that the Basys3 USB drivers have been correctly installed.
